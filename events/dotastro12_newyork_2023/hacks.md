# Hacks from .Astronomy 12

.Astronomy 12 was held at the Center for Computational Astrophysics (CCA), Flatiron Institute, New York, 3–5 October 2023.

_Full hack list to be added. Known GitHub repos from this event are listed below._

## Known Projects and Tutorials

### Contributing to Astropy
**Organisers:** Elliott Molter  
**GitHub:** [emolter/dotAstro12-contribute-to-astropy](https://github.com/emolter/dotAstro12-contribute-to-astropy)  
A session on how to contribute to the Astropy project.

### Snakemake and Show Your Work
**Organisers:** Sam Vaughan  
**GitHub:** [samvaughan/dotastro-snakemake-showyourwork-session](https://github.com/samvaughan/dotastro-snakemake-showyourwork-session)  
A tutorial session on reproducible workflows using Snakemake and the showyourwork framework.

### Multi-Order Coverage (MOC) Hack
**Organisers:** David Rodriguez  
**GitHub:** [dr-rodriguez/MOC_dotAstro12](https://github.com/dr-rodriguez/MOC_dotAstro12)  
Working with Multi-Order Coverage maps for astronomical footprints.

### dotAstro Demo
**GitHub:** [eblur/dotAstro-demo](https://github.com/eblur/dotAstro-demo)  
Demo materials from .Astronomy 12.
