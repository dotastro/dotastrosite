## Hacks

[1 hack documented](hacks.md)

Known hacks from this event:
- **Chromotone** (Jon Yardley, Chris North, Haley Gomez, Edward Gomez): Exploring multiwavelength astronomy with sound. Each waveband has a pitch; the brightness in that waveband is represented by the volume of a note. Scaning a mouse over an image plays a chord where components correspond to the intensity at each pixel. This .Astronomy 3 version aimed to make a lighter, more web-friendly version with added functionality. [Source](https://github.com/jonyardley/Chromotone)
