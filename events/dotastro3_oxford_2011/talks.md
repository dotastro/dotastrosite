# Talks

<p class="page-lead">4-6 April 2011, New College, Oxford</p>

## Trending Topics

Amazon AWS, Crowdsourcing, Twitter, SETI

## Notes

The full talk list for .Astronomy 3 is still being reconstructed. The archive pages from the original site noted that detailed session and talk records were "still in the process of being collated" even before the site was lost.

From the liveblog and conference reports, we know the event included discussions of:
- Citizen science platforms and the Zooniverse
- Amazon Web Services for astronomy
- Chromoscope and browser-based visualisation
- REF impact evaluation for astronomy tools
- Career paths for astronomers leaving academia
