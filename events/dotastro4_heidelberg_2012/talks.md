## Talk Highlights

Kevin Govender's talk on using astronomy for development was particularly well-received, with extensive audience discussion about the role of astronomy in society. The talk covered the IAU Office for Astronomy Development's mission to use astronomy as a tool for development, particularly in Africa and the developing world.

Michelle Borkin demonstrated how Amazon's Mechanical Turk could be used for astronomical classification tasks, and showed the Glue visualisation tool for multi-dimensional data exploration.

Julie Steele and Noah Iliinsky delivered an engaging session on data visualisation principles, drawing from their O'Reilly publications.

Alyssa Goodman's Seamless Astronomy talk discussed linking data, literature, and visualisation tools, with live demonstrations of how different software could communicate via SAMP.
