# Hacks from .Astronomy X

.Astronomy X was held at the Space Telescope Science Institute, Baltimore, 24–27 September 2018.

## Hacks

### Alternatives to Transparency in Multiwavelength Visualization
**Creator:** Carolina Odman  
Exploring other ways to browse overlaid images in different wavelengths beyond simple transparency.

### Bokeh Hacking with FITS Images
**Creator:** David Rodriguez  
Working out how to use FITS images, catalog data, and observation footprints with the Bokeh visualisation library.  
[Source](https://github.com/dr-rodriguez/BokehHack) | [Live](https://dr-portal.herokuapp.com)

### Contributor Plots for Astropy-Affiliated Packages
**Creators:** Matthew Bourque, Erik Tollerud  
Code that uses the GitHub REST API to plot the number of unique contributors over time for astropy-affiliated packages.  
[Source](https://github.com/astropy/astropy-procedures/pull/96)

### Data Sonification
**Creators:** Clara Brasseur, Nuwanthika Fernando  
Experimenting with turning astronomical data into sound.  
[Source](https://github.com/ceb8/SonificationExperiments)

### Developing Social Justice Reading Groups for Astronomical Communities
**Creators:** Lauren Chambers, Gourav Khullar  
Collecting resources and texts to generate a syllabus for social justice/equity/diversity/inclusion discussion groups in astronomical institutions.

### Optical Depth in a Minute
**Creators:** Rocio Kiman, Carolina Odman  
Creating short (approximately 60 second) educational videos that explain astronomical concepts using simple tools.

### Public Outreach Tool for Exploring Research Interactively
**Creators:** Claire Murray, Jules Fowler  
A web-based outreach tool for contextualising astronomical research for the public by tagging recent science results to interactive all-sky maps, using Aladdin Lite.  
[Source](http://github.com/cmurray-astro/dotastro_hack)

### Space, Wow
**Creators:** Joseph Long, Frank Summers  
Building something like the HubbleSite video experience live in the browser.  
[Source](https://github.com/joseph-long/space-wow) | [Live](https://joseph-long.github.io/space-wow/)

### Specreduce: A Hack Day Package
**Creators:** Azalee Bostroem, Mees Fix, Iva Momcheva, Laura Prichard  
Advancing the specreduce package: clean wavelength tables, convenient function access, air-to-vacuum conversion, and testing of the trace function.  
[Source](https://github.com/astropy/specreduce)

### You Might Also Like These Matplotlib Keywords
**Creator:** Benjamin Weiner  
Reads a training set of Python files for matplotlib function calls, then reads your file and shows the most commonly used keywords for the functions you actually used.  
[Source](https://github.com/bjweiner/dotastro/tree/master/keyword_recommend)
