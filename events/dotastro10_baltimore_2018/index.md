The tenth .Astronomy conference, known as ".Astronomy X" (using the Roman numeral), was held at the Space Telescope Science Institute in Baltimore, Maryland. STScI is the science operations centre for the Hubble Space Telescope and the James Webb Space Telescope, making it a fitting venue for a conference at the intersection of astronomy and technology.

## Hacks

[9 hacks documented](hacks.md)

Hacks from this event include:
- **Alternatives to transparency in multiwavelength visualization** (Carolina Odman): Exploring other ways to browse overlaid images in different wavelengths.
- **Bokeh Hacking with FITS Images** (David Rodriguez): Working out how to use FITS images, catalog data, and observation footprints with Bokeh. [Source](https://github.com/dr-rodriguez/BokehHack)
- **Contributor plots for astropy-affiliated packages** (Matthew Bourque, Erik Tollerud): Using the GitHub REST API to plot unique contributors over time for astropy-affiliated packages. [Source](https://github.com/astropy/astropy-procedures/pull/96)
- **Data Sonification** (Clara Brasseur, Nuwanthika Fernando): Experimenting with turning data into sound. [Source](https://github.com/ceb8/SonificationExperiments)
- **Developing Social Justice Reading Groups for Astronomical Communities** (Lauren Chambers, Gourav Khullar): Collecting resources to generate a syllabus for social justice/equity/diversity/inclusion discussion groups in astronomical institutions.
- **Optical Depth in a minute** (Rocio Kiman, Carolina Odman): 60-second educational videos made with simple tools.
- **Public outreach tool for exploring research interactively** (Claire Murray, Jules Fowler): A web-based outreach tool for contextualizing astronomical research for the public, using Aladdin Lite and interactive all-sky maps.
- **Space, Wow** (Joseph Long, Frank Summers): Building a Hubble-style video experience in the browser. [Source](https://github.com/joseph-long/space-wow)
- **Specreduce: a hack day package** (Azalee Bostroem, Mees Fix, Iva Momcheva, Laura Prichard): Advancing the specreduce package. [Source](https://github.com/astropy/specreduce)
- **You Might Also Like These Matplotlib Keywords** (Benjamin Weiner): Reads your Python file for matplotlib calls and shows the most commonly used keywords for the functions you used. [Source](https://github.com/bjweiner/dotastro/tree/master/keyword_recommend)
