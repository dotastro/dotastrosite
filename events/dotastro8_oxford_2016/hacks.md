# Hacks from .Astronomy 8

.Astronomy 8 was held at Pembroke College, Oxford, 20–23 June 2016. The Hack Day took place on Wednesday 22nd June 2016.

## Hacks

### .draft
**Creator:** Andy Casey  
Automagically highlight LaTeX differences between GitHub drafts.  
[Live](https://gentle-cove-44775.herokuapp.com/)

### StingRA: Simbad Quick-Look Chrome Extension
**Creators:** Hugh Osborn, Jonathan Fay  
A Chrome extension that lets astronomers highlight an astronomical object and open a quick-look pop-up of magnitudes, images, sky position, and links to SIMBAD and WorldWide Telescope.  
[Source](http://github.com/astrojonathan/StingRA) | [Install](https://chrome.google.com/webstore/detail/stingra/jopoiaekgcofgjjlmcckfnafbjpedeim)

### Are You Still Talking?
**Creator:** Edward Gomez  
Use a blink(1) light to control your speakers when chairing a session. A Python package that installs a command-line talk timer, green while talking, orange in the warning period, red when time's up.  
[Source](http://github.com/zemogle/session_chair)

### arxiv59
**Creator:** Andy Casey  
Twitter bot that posts papers which missed the top-of-the-arXiv deadline by about a second.  
[Source](https://github.com/andycasey/arxiv59) | [Live](https://twitter.com/arxiv59)

### The ArXiv Mailer
**Creator:** Dan Foreman-Mackey  
An app to modernise the formatting of the daily arXiv email.  
[Source](https://github.com/dfm/ArXivMailer) | [Live](http://arxiv.dfm.io)

### AstroBadger
**Creators:** Matthew Kenworthy, Lisa Ballard  
Getting you in the habit of writing 200 academic words per day, every day. Gamified writing! Chrome plugin + Flask app.  
[Source](https://github.com/basilleaf/astrobadger)

### AstroGoldStars
**Creators:** Geert Barentsen, Kelle Cruz  
Give a Gold Star to a fellow scientist by tweeting @username++ at @AstroGoldStars on Twitter. Solves credit sharing in science.  
[Source](http://github.com/barentsen/AstroGoldStars) | [Live](https://twitter.com/AstroGoldStars)

### bvi-sonification
**Creators:** Coleman Krawczyk, Jamie Ferguson, Andy Newsam, Adam Avison, Philipp Plewa  
Uses a Leap Motion device to allow a user to listen to a galaxy's spectra based on hand position.  
[Source](https://github.com/jamieferguson94/bvi-sonification/tree/master)

### Fakespeare
**Creators:** Ruth Angus, Dan Foreman-Mackey  
A LSTM neural network trained on the corpus of all Shakespeare plays that can generate hilarious and nonsensical plays.  
[Source](https://github.com/RuthAngus/Fakespeare)

### Galaxy Zoo Maze
**Creators:** James Gilbert, Ricarda Beckman, Ellen Schallig, Becky Smethurst, Marten Veldthuis  
Creates a responsive view of a Zooniverse subject and a QR code, then supports scanning and reacting to results.  
[Source](https://github.com/chrislintott/GZMaze) | [Live](http://gzmaze.herokuapp.com)

### GitNL
**Creators:** Brett Andrews, Brian Cherinka, Erik Tollerud  
Run Git commands using normal English sentences parsed with Google's natural language processor.  
[Source](http://github.com/eteq/gitnl)

### Hacks Collector
**Creators:** Erik Tollerud, Daina Bouquin, Tom Robitaille, Edward Gomez, Becky Smethurst, Arna Karick  
A way to archive all hacks made at .Astronomy conferences. Each hack has a separate YAML file containing all metadata.  
[Source](http://github.com/dotastro/hacks-collector) | [Live](https://dotastro.github.io/hacks-collector/)

### Markov Papers
**Creator:** Benjamin Pope  
Generates paper titles/abstracts with a Markov chain trained on real papers returned by any ADS search keywords.  
[Source](https://github.com/benjaminpope/markovpapers)

### Meta^4hack
**Creators:** Erik Tollerud, Daina Bouquin, Tom Robitaille  
Metadata on a hack about metadata on hacks, the submission form for this hack list.  
[Source](https://github.com/dotastro/hack-list-submission-app) | [Live](https://dotastronomy-hack-submitter.herokuapp.com)

### Project Apollo Stories
**Creators:** Arna Karick, Sam Vaughan  
A Zooniverse Project to re-create incredible stories from the Apollo missions using selected images from the Apollo Project Flickr gallery and Timeline JS.  
[Source](https://github.com/samvaughan/ApolloProject) | [Live](https://www.zooniverse.org/projects/projectapollo-dotastro8/project-apollo-stories)

### SIMBAD the Bot
**Creator:** Simon Murphy (@AstroSmurph)  
Twitter bot that tweets a multi-wavelength animated GIF of a random SIMBAD object a few times a day.  
[Source](https://github.com/simonmurphy/simbad-the-bot) | [Live](https://twitter.com/simbad_the_bot)

### Spacebar
**Creator:** Scott Thomas  
Astronomy-themed mini-games where you only need the spacebar. Keep rain out of the dome while observing, find the galaxies, and more.  
[Source](http://github.com/swt30/spacebar)

### Stellar Autopsies Zooniverse
**Creators:** Adam A. Miller, Brigitta Sipocz, Ashley Villar  
A Zooniverse Project to classify the light curves of dying stars.  
[Source](https://github.com/bsipocz/stellar-autopsies) | [Live](https://www.zooniverse.org/projects/adamamiller/stellar-autopsies)
