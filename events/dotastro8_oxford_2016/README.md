# .Astronomy 8 — Oxford, 2016

**Date:** June 2016
**Location:** Pembroke College, University of Oxford, UK
**Theme:** Open science, hardware hacking, citizen science, LSST, library archives, ESA Sky
**Attendance:** ~65 people
**Organisers:** Becky Smethurst (lead), Chris Lintott, Sarah Kendrew. LOC: Mark Richardson, Edward Gomez, Ruth Angus, Sam Vaughan, Ellen Schallig, Sandor Kruk

## About

The eighth .Astronomy returned to Oxford, this time at Pembroke College. Organised by Becky Smethurst, then an Oxford PhD student, with support from Chris Lintott and Sarah Kendrew.

Talks covered open science in the Kepler missions, hardware hacking, citizen science, LSST data schools, library archives in astronomy, and ESA Sky. Speakers included Adam Miller, Carl Ferkinhoff, Ali Swanson, Geert Barentsen, James Gilbert, Tom Robitaille, Sarah Kendrew, and Daina Bouquin. The pre-dinner talk was by Aleks Scholz about WETI.

Day 0 featured tutorials on programming languages, GitHub, web hosting, Flask, D3, web scraping, Django, and argparse.

The Hack Day took place on Wednesday 22nd June 2016. All hacks are listed on the .Astronomy 8 hack collector page.
